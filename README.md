# Cell Colonies

## Project Info

Cell Colonies is a simulation game where different cells interact, compete, and shape the outcome of a living grid. Built entirely solo as my Grade 12 CS final project, my game combines original systems, mechanics, visuals, and design ideas.
Every feature, concept, and piece of artwork was created by me. Music and sound effects were obtained from free-to-use music websites and are properly credited.

## Features

- Battle through a living grid where rival colonies fight to survive
- Replay the same world with unique seeds and outcomes, able to pause, step, fast-forward, and rewind at any time
- Discover unique cell types with different roles, behaviours, and interactions
- Watch colonies pathfind, collide, and adapt
- Rewind through the timeline to inspect exactly how the simulation evolved
- Customize world and performance settings
- Zoom, pan, inspect cells, and toggle debug overlays for a deeper view of your simulation
- Personalize controls with changable controls
- Track, regenerate, copy, and paste seeds to share or replay worlds
- Learn the game through a guided tutorial with Bob Ross-style dialogue
- Tune fullscreen, resolution, audio, and performance options for your setup

## Installation + Running
After downloading the game, it should be as running `main.py` in the project root:
```
# ../cell-colonies/
python main.py
```
Be sure to install Python 3.14.2.
